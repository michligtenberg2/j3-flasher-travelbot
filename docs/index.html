Travelbot Flasher

This site hosts documentation for the Travelbot flasher tool.

The application flashes a Samsung Galaxy J3 (SM-J320FN) with a minimal LineageOS ROM and provides a GUI built with PyQt6.

Browse the Frequent issues tab for help with common problems such as FRP lock.

See Root info to learn when root access might be useful.
