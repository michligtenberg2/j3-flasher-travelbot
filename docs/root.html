Heb ik root nodig voor Travelbot?

Nee – voor Travelbot is rooten niet noodzakelijk. Je kunt vrijwel alle functies gebruiken zonder roottoegang.

✅ Wat werkt zonder root?

FunctieVereist root?
📍 Locatie uitlezen via GPS❌ Nee
🔊 Tekst-naar-spraak uitvoeren❌ Nee
🌐 Reacties ophalen via backend (AI)❌ Nee
🎵 Reacties afspelen via TTS of MP3❌ Nee
🧭 In foreground-service draaien❌ Nee
🧑‍💻 Gebruikersinterface en knoppen❌ Nee
🚗 Automatisch opstarten bij boot❌ Nee (mogelijk via manifest)

Travelbot is ontworpen om te draaien op oude Android-telefoons zonder root.

🔐 Wanneer is root wél nuttig?

Alleen als je geavanceerde systeemacties wilt zoals:

  • Vliegtuigmodus aan/uit zetten
  • Andere apps afsluiten of starten
  • Meldingen van andere apps blokkeren
  • Travelbot als systeemapp installeren
  • TTS-engine vervangen of manipuleren op systeemniveau

Voor de meeste gebruikers is dit niet nodig.

🧠 Waarom TWRP dan wél?

TWRP (TeamWin Recovery Project) gebruik je alleen om:

  • LineageOS of een lichte custom ROM te installeren
  • Travelbot eenvoudig te installeren
  • FRP/OEM locks te omzeilen
  • Apparaten te "recyclen" voor hergebruik

TWRP ≠ root, maar maakt rooten mogelijk voor wie dat wíl.

📌 Advies

Gebruik root alleen als je weet wat je doet, of specifieke systeemtoegang nodig hebt.
Travelbot werkt standaard zonder root. De root-tab in de GUI is optioneel.
